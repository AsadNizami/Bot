# Instagram Scroll Bot
## Installation
```bash 
  pip install -r requirements.txt
```

### Configure the credentials.py file
```bash 
  username = <username_here>
  password = <password_here>
```
## Usage

```bash 
  python scrollbot.py --total_like <Total number of post to like>
```
